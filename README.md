# Ecommerce_Olist_Store_Project
The Olist Store E‑commerce Project is a popular dataset and case study used in analytics and data science to explore how online marketplaces operate. Olist is a Brazilian e‑commerce platform that connects small businesses to major online marketplaces, enabling them to sell products nationwide.
- Objective: The project focuses on analyzing customer behavior, order management, product performance, and seller efficiency to improve marketplace operations.
- Dataset: It contains detailed information on orders, products, sellers, customers, payments, reviews, and geolocation.
